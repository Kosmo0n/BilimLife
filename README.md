# BilimLife
Интерактивный прототип платформы для изучения языков.

## Технологии
- React
- Tailwind CSS
- Framer Motion (Animations)
- Lucide Icons

## Как запустить
1. Установите все зависимости из package.json:
   ```bash
   npm install
   ```
2. Запустите проект (команда `start` теперь настроена на Vite):
   ```bash
   npm start
   ```
